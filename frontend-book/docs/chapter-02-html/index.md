# Chapter 2: HTML

<p class="byline" markdown>Muhammad Umar Sajid · Trainer at Bano Qabil Sahiwal · Front End Development and Graphic Design</p>

HTML is the skeleton of every web page. It holds the headings, the paragraphs,
the images, the links, and the forms. Before a page can look good or do anything
clever, it needs this structure underneath.

This chapter starts with the big picture of the three web languages, then has you
write your first real HTML page in VS Code. From there you add text, lists, links,
and images, build a contact form that looks like the real thing, and finally learn
to give your pages meaning so search engines and screen readers understand them.

!!! tip "Type the code yourself"
    Reading HTML is easy. Writing it is the skill. Type every example into your
    own file and open it in the browser. You learn far more from your own typos
    than from clean code on a screen.

## Mega assignment

!!! bq-assignment "Hand-code your client's page structure, no styling yet"
    Structure first. Build the full skeleton of your client's one-page site in
    pure, semantic HTML. It will look plain, and that is exactly right. A strong
    skeleton is what makes every later chapter easy.

    **What you build**

    - One `index.html` with: a header and nav, a hero with the business name and one line about it, a section for what they offer (a list or a menu), an about section, an image, a contact section with a real form (name, message, a way to reach them), and a footer.
    - Every section in the **right semantic tag**, every image with **alt text**, every input with a **matching label**.

    **Done when**

    - [ ] The page uses `header`, `nav`, `main`, `section`, and `footer`, not a pile of `div`s.
    - [ ] Every image has meaningful alt text.
    - [ ] Every form input has a label tied to it.
    - [ ] The page passes the [W3C validator](https://validator.w3.org/) with zero errors.

    **Stretch goal:** Add a second page (a full menu or a services page) and link the two together in the nav.
