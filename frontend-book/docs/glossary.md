# Glossary

Every technical term used in this book, in plain English with an Urdu note where
it helps. Terms are added here as each lesson is written. Use the search box to
jump to a term, or browse A to Z below.

## A

### active learning
Learning by doing the thing yourself, not just watching or reading.
First introduced in lesson 0.4.

**Roman Urdu:** khud kar ke seekhna

First introduced in lesson 0.4.

**Roman Urdu:** yaad se jawab nikalna, dobara parhne ke bajaye
