# Chapter 3: Adobe Photoshop

<p class="byline" markdown>Muhammad Umar Sajid · Trainer at Bano Qabil Sahiwal · Front End Development and Graphic Design</p>

Photoshop is the tool people use to edit photos and paint raster art. It is paid, but it has a 7-day free trial. There is also a free browser tool called Photopea that works almost the same way. So you can follow this whole chapter without paying anything.

*(Roman Urdu: Photoshop woh tool hai jisse log photos edit karte hain aur raster art banate hain. Yeh paid hai, lekin Photopea ek free browser tool hai jo lagbhag waise hi chalta hai. Is liye tum poora chapter bina paise diye seekh sako ge.)*
