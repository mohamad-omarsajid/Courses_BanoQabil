# 0.1 Your Computer, Files & Cloud Drives

!!! info "Why this lesson?"
    By the end of this lesson you can keep your design files tidy, pick the
    right image type for each job, and back your work up to Google Drive. These
    are small habits, but they save you a lot of pain later.

    *(Roman Urdu: Is sabaq ke baad tum apni design files theek tareeqe se rakh
    sako ge, har kaam ke liye sahi image type chun sako ge, aur apna kaam
    Google Drive par mehfooz kar sako ge.)*

## What you will learn

- How files and folders work for designers
- The main image types and when to use each
- What a layered file is
- Google Drive basics: upload, folders, and share links

## Files and folders

A **file** (Roman Urdu: ek cheez jo computer par save hoti hai, jaise ek photo
ya logo) is one saved thing. A poster you make is a file. A photo is a file.

A **folder** (Roman Urdu: ek dabba jis mein files rakhi jati hain) holds files.
You put files inside folders so you can find them later. A folder can hold
other folders too.

Designers make many files. One project can have ten or more. So good folders
matter. Think of it like a shop with clean shelves. You always know where each
thing is.

!!! tip "Turn on file extensions first"
    Windows hides the file type by default. Open any folder, click **View**,
    and turn on **File name extensions**. Now you can see the `.jpg` or `.png`
    part of each name. You need this for the whole course.

## The main image types

Every image file ends with a short type word, like `.jpg` or `.png`. Each type
is good for a different job. Here are the four you will use most.

- `.jpg` is for **photos** (Roman Urdu: asli tasveerein, jaise camera se li gayi
  photo). It makes small files, so it is great for pictures with many colors.
  It cannot have a see-through background.
- `.png` is for **logos and graphics with a transparent background** (Roman
  Urdu: aisi image jiska background see-through ho, taake woh kisi bhi rang par
  saaf baithe). Use `.png` when you need clean edges and no white box behind.
- `.svg` is for **vectors** (Roman Urdu: math lines se bani art jo har size par
  saaf rehti hai). A logo saved as `.svg` stays sharp at any size, small or
  huge. It never gets blurry.
- `.pdf` is for **print and sending** (Roman Urdu: ek file jo har computer par
  bilkul same dikhti hai, print ke liye behtareen). Send a `.pdf` to a print
  shop or a client. It looks the same everywhere.

!!! example "Pick the right type"
    - A food photo for Instagram? Save it as `.jpg`.
    - A shop logo to put on top of a colored banner? Save it as `.png`.
    - A logo the print shop will blow up to a big sign? Give them an `.svg`.
    - A finished menu to email to the client? Send a `.pdf`.

!!! warning "A photo is not a vector"
    If you make a logo from a photo, it will get blurry when made bigger. A real
    vector logo (`.svg`) stays crisp. Try to keep logos as vectors when you can.

## What is a layered file?

While you work, you keep all your pieces apart. The text, the photo, and the
background each sit on their own layer. A **layer** (Roman Urdu: ek see-through
sheet jis par design ka ek hissa hota hai) is like a clear sheet stacked on top
of others.

A **layered file** (Roman Urdu: aisi file jisme saare hissay alag alag mehfooz
hote hain) keeps all those sheets editable. Examples are `.psd` from Photoshop
and the editable file in Figma or Canva.

Here is the key habit:

1. Keep one layered file while you work, so you can change things later.
2. When you are done, export a flat file like `.jpg`, `.png`, or `.pdf` to
   share.

A flat file is small and easy to send. The layered file is your master copy.
Never delete the layered file. You will thank yourself later.

## Google Drive basics

**Google Drive** (Roman Urdu: internet par aapki files rakhne ki jagah, jise
kisi bhi device se khol sakte ho) keeps your work safe online. If your laptop
breaks, your files are still there.

To upload a file:

1. Open `drive.google.com` and sign in with your Google account.
2. Click **New**, then **File upload**.
3. Pick the file from your computer and wait for it to finish.

To make a folder in Drive:

1. Click **New**, then **Folder**.
2. Give it a clear name, like `banoqabil-design`.
3. Drag your files into it.

To share your work with a link:

1. Right click the file or folder, then click **Share**.
2. Under **General access**, choose **Anyone with the link**.
3. Click **Copy link** and send it on WhatsApp or email.

!!! warning "Be careful with share links"
    A link set to **Anyone with the link** can be opened by anyone who has it.
    Only share private work with people you trust.

!!! tip "tl;dr"
    Files hold your work and folders keep them tidy. Use `.jpg` for photos,
    `.png` for transparent logos, `.svg` for sharp vectors, and `.pdf` for
    print. Keep a layered master file while you work. Back everything up to
    Google Drive.
