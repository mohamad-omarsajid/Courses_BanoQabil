# Chapter 0: Getting Ready

<p class="byline" markdown>Muhammad Umar Sajid · Trainer at Bano Qabil Sahiwal · Front End Development and Graphic Design</p>

Before you open any design tool, let us get comfortable with the computer
itself. You will learn how files work, what the main image types are, and how
to keep learning design on your own.

If you have used a phone but never really used a laptop, this chapter is for
you. Nothing here is hard. Take it slow.

*(Roman Urdu: Koi design tool kholne se pehle, hum computer, files, image
types aur design seekhne ke aasaan tareeqe samajh lete hain. Ghabrane ki koi
baat nahi.)*

## Lessons in this chapter

1. [Your Computer, Files & Cloud Drives](0-1-computer-files-cloud.md) - files, image types, and Google Drive for designers.
2. [Word, Excel & Google Search for Designers](0-2-word-excel-google-designers.md) - moodboards, invoices, and finding images you may use.
3. [AI as a Design Brain](0-3-ai-as-design-brain.md) - ChatGPT and Claude for ideas, names, copy, and honest critique.
4. [Learning to Learn (designer edition)](0-4-learning-to-learn-design.md) - a visual diary, the redraw exercise, and good feedback.

!!! tip "No rush"
    There is no clock and no deadline. Finish a lesson, take a break, come back
    later. The lessons will wait for you.
