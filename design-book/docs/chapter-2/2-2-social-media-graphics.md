# 2.2 Social Media Graphics

!!! info "Why this lesson?"
    By the end of this lesson you can make posts, stories, and reel covers in the
    right sizes, and keep a set of posts looking like they belong together.

    *(Roman Urdu: Is sabaq ke baad tum sahi size mein posts, stories, aur reel
    covers bana sako ge, aur ek set ko aisa rakh sako ge ke sab milte julte lagein.)*

## What you will learn

- The right sizes for an Instagram post, story, and reel cover
- What aspect ratio means
- The difference between a square and a tall shape
- How to keep a set of posts matching

## Social media sizes

Each spot on Instagram wants a different shape. If you use the wrong size, your
design gets cut off or looks small. Use these common sizes:

- Instagram post: `1080 x 1080` pixels, a square.
- Instagram story: `1080 x 1920` pixels, tall.
- Instagram reel cover: `1080 x 1920` pixels, tall.

You do not need to remember these numbers. In Canva, just search "Instagram post"
or "Instagram story" and it sets the right size for you.

!!! tip "Let Canva pick the size"
    On the dashboard, type the name of what you want, like "Instagram story".
    Canva opens a blank design in the correct size. You do not type the pixel
    numbers yourself.

## What is an aspect ratio?

The **aspect ratio** (Roman Urdu: design ki chaurai aur unchai ka aapas mein
rishta) is the shape of your design. It compares the width to the height. It does
not care about the exact size, only the shape.

Two common ones:

- `1:1` is a square. The width and height are equal. This is the normal feed post.
- `9:16` is tall. The height is much more than the width. This is a story or a
  reel cover.

Think of it this way. A `1:1` design fits a square box. A `9:16` design fits the
full tall screen of a phone. Same idea, different shape.

!!! warning "Wrong ratio gets cut"
    If you make a square design and upload it as a story, Instagram may cut the
    top and bottom, or leave empty bars. Pick the right ratio before you start,
    not after.

## Keep a set matching

A single nice post is good. A set of posts that match looks like a real brand.
This is how shops and creators look neat on their page.

To keep a set matching:

1. Use the same two or three colors in every post.
2. Use the same font for headings in every post.
3. Put your logo in the same corner each time.
4. Keep the same kind of photos, for example all bright, or all dark.

!!! example "Matching set, different words"
    Make one post you like. Then make a copy of it. Change only the words and the
    main photo. Keep the colors, font, and logo the same. Now you have two posts
    that clearly belong together.

A brand kit helps a lot here. It saves your colors and fonts, so picking the same
look each time is fast.

!!! tip "tl;dr"
    Each Instagram spot wants its own size. Let Canva set it by searching the
    name. The aspect ratio is the shape: `1:1` is square, `9:16` is tall. To make
    a set match, reuse the same colors, font, and logo in every post.

## Optional assignment

Make a set of three matching Instagram posts. They can be about anything you
like, for example three tips, or three favorite foods. Use the same two colors,
the same heading font, and put your logo in the same corner on all three. Change
only the words and the photo on each.
