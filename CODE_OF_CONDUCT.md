# Code Of Conduct

This project should be a safe, respectful place for learners, contributors, and maintainers.

## Expected Behavior

- Be kind and patient with beginners.
- Give feedback that helps the work improve.
- Assume good intent when someone makes a mistake.
- Use welcoming language.
- Respect different backgrounds, skill levels, and communication styles.

## Unacceptable Behavior

- Harassment, insults, threats, or personal attacks.
- Discrimination or exclusionary language.
- Sharing private information without permission.
- Repeated off-topic comments after being asked to stop.
- Any behavior that makes the community less safe for learners.

## Reporting

If you see a problem, contact the maintainers through GitHub Issues or email:

```text
hello@banoqabilsahiwal.org
```

Maintainers may remove comments, close issues, or block users to protect the community.
